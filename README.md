# NoteKeeper
用Java语言开发的安卓简单demo：任务管理器

#### 目的
 - 熟悉ConstraintLayout安卓布局系统
 - 使用Parcelable来不同activity中传输数据(NoteActivity与NoteListActivity)
 - 熟悉安卓系统中的显示与隐式Intent
 - 了解安卓生命周期
 
#### 功能描述
 页面由主界面的ListView与详情页构成，点击每个数据条目可以进入具体的详情页面。详情页面数据可以编辑，点击返回键数据会自动保存，详情页右上方按钮包含发送邮件的功能，可以发送邮件
